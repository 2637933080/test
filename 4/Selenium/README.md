# Selenium Login Test

This project contains a simple login page and a Selenium test script to test it with different browsers.

## Prerequisites

- Python 3.7+
- Selenium: `pip install selenium`
- ChromeDriver: Download from https://chromedriver.chromium.org/downloads and place in PATH or specify path in script.
- GeckoDriver for Firefox: Download from https://github.com/mozilla/geckodriver/releases and place in PATH or specify path in script.

## Files

- `login.html`: Simple login page
- `test_login.py`: Selenium test script

## Running the Test

Run `python test_login.py`

It will test with Chrome and Firefox.
